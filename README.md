# Learning C
Repository with exercises and practical lessons in C Language from the 1nd semester of the Systems Development course.
## Lessons
| Folder | Content | Topics learned |
|--------|---------|----------------|
| `aula01/` | Basic concepts | variables, `float`, `if/else if`, `scanf`, `printf` |
| `aula02/` | Loop syntax | nested `for`, decrement, increment, pattern logic |
| `aula03/` | Arrays and strings | `int[]`, `char[]`, `fgets`, `strlen`, `scanf`, loops with arrays |
| `aula04/` | Functions | `float`, `void`, parameters, return, refactoring |
| `aula05/` | Pointers | `*`, `&`, memory address, pass by reference |
| `aula06/` | Structs | `typedef`, `struct`, array of structs, functions with structs |

---

## How to run
```bash
gcc aula01/main.c -o programa
./programa
```

## C++

Exercises from the introduction to C++ — loops, conditionals, and logic applied to classic programming problems.

| File | Description | Topics |
|------|-------------|--------|
| `C++/01ex.cpp` | Check if a number is prime | `bool`, `for`, modulo, `break` |
| `C++/02ex.cpp` | Powers of 2 from 2⁰ to 2ⁿ | `long long`, `for`, multiplication |
| `C++/03ex.cpp` | Sum numbers until a negative is entered | `while`, `break`, accumulator |
| `C++/04ex.cpp` | Find the smallest and largest of 10 numbers | `for`, `if/else`, min/max logic |
| `C++/05ex.cpp` | List multiples of a number up to a limit | `for`, modulo |
| `C++/06ex.cpp` | Multiply numbers until their sum exceeds 100 | `while`, `break`, accumulator |
| `C++/07ex.cpp` | Calculate the average height of N people | `for`, `float`, average |
| `C++/08ex.cpp` | Sum of all even and odd numbers up to N | `for`, modulo, two accumulators |
| `C++/09ex.cpp` | Fibonacci sequence up to a number (50–100) | `while`, sequence logic, input validation |
| `C++/10ex.cpp` | Number guessing game with hints and attempt count | `srand`, `rand`, `for`, `while`, input handling |

### How to compile (C++)

```bash
g++ C++/01ex.cpp -o ex01
./ex01
```
